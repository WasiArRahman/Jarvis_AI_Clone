# Jarvis AI Clone

## Description
Jarvis AI Clone is an artificial intelligence project that simulates basic conversational abilities and tasks.

## Installation
To install the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/WasiArRahman/Jarvis_AI_Clone.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Jarvis_AI_Clone
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the project, execute the following command:
```bash
python jarvis.py
```
